# py_logger
logger_on_python
Этот логер был написан исключительно в образовательных целях на python для комьюнити,
дабы продемонстрировать возможности языка в профессиональных целях.
Данный код не видим антивирусу при компиляции через pyinstaller и подлежит обфускации через
pyarmor. Что позволяет использовать его для улучшения антивирусных программ а так же написания своих.
!!!СТРОГО В ОБРАЗОВАТЕЛЬНЫХ И ПОЛЕЗНЫХ ЦЕЛЯХ!!!

-----------------------------------------------------------------------

This logger was written exclusively for educational purposes in python for the community
in order to demonstrate the capabilities of the language for professional purposes.
This code is not visible to the antivirus when compiled via pyinstaller and is subject to obfuscation via
pyarmor. Which allows you to use it to improve antivirus programs as well as write your own.
